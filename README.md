# PDF_chatbot

## Description
This project is a Generative AI application that allows users to upload PDF documents and ask questions about their content using Retrieval-Augmented Generation (RAG).

## Technologies Used
- Python
- Streamlit
- LangChain
- ChromaDB
- OpenAI API
- PyPDF

## Features
- Upload PDF
- Ask questions
- AI-generated answers
- Vector search with ChromaDB

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Add your OpenAI API key in a `.env` file
4. Run:
   streamlit run app.py
